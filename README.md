# Analyse des best-sellers Amazon France

## Objectif du projet

Ce projet analyse les best-sellers Amazon France afin d’identifier les catégories dominantes, les tendances de vente et les caractéristiques des livres les mieux classés.

L’objectif est de transformer des données brutes en informations exploitables pour mieux comprendre les dynamiques du marché du livre en France.

## Compétences mobilisées

- Nettoyage de données
- Analyse exploratoire
- Catégorisation des livres
- Analyse des tendances
- Visualisation des données
- Création de tableau de bord

## Outils utilisés

- Python
- Pandas
- Matplotlib
- Looker Studio
- Excel / CSV

## Étapes du projet

1. Importation des données
2. Nettoyage et structuration des colonnes
3. Harmonisation des catégories de livres
4. Analyse des ventes, prix, notes et avis
5. Création de visualisations
6. Construction d’un tableau de bord final

## Analyses réalisées

- Répartition des livres par catégorie
- Évolution des tendances de vente
- Analyse des prix moyens
- Analyse des notes et des avis
- Identification des catégories les plus représentées

## Résultats principaux

Ce projet permet de mieux comprendre quels types de livres dominent les meilleures ventes Amazon France et quelles caractéristiques reviennent le plus souvent parmi les ouvrages les mieux classés.
